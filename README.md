# 💳 Debt Pressure System

A modern, powerful web application for managing and tracking your debts with intelligent payoff strategies.

## 🚀 Features

✅ **Debt Management** - Add, edit, and delete debts  
✅ **Payoff Strategies** - Snowball & Avalanche algorithms  
✅ **Advanced Analytics** - 6+ interactive charts  
✅ **Pressure Score** - Quantified debt stress indicator  
✅ **Interest Calculations** - Real-time interest projections  
✅ **Mobile Responsive** - Works on all devices  
✅ **Modern Design** - Beautiful gradient UI with animations  
✅ **Local Storage** - Your data stays on your device  

## 📦 Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Zustand** - State management
- **Recharts** - Data visualization
- **Framer Motion** - Animations
